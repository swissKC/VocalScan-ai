# VocalScan AI 🎤

AI-powered singing voice analyzer that evaluates pitch, vocal range, and tone quality from uploaded audio files.

## Overview

VocalScan AI is a web application that analyzes singing voice using artificial intelligence and audio signal processing.  
Users can upload an MP3 audio file and the system will analyze the vocal performance.

## Features

- Upload singing audio file (MP3)
- Detect pitch (เสียงสูงต่ำ)
- Analyze vocal range
- Evaluate tone stability
- Simple web interface
